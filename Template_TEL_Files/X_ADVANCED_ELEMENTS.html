Advanced elements

Please note: the following elements can only be updated using HTML and therefore require basic coding knowledge. If you are unable to edit them yourself but would like to use any of the elements in your content, please contact TEL for support.

Use this page to copy & paste individual elements in to your own content.

Please read the instructions below and/or watch the video demo beneath to learn how to add any element to your own pages.

Important: you must be viewing this page in 'edit mode' to copy page elements

To copy an element correctly:

  • Make sure you are viewing this page in 'edit mode' (i.e. you can see the editing toolbar at the top of the window)
  • Scroll down to the element you wish to copy
  • Highlight everything from START COPY HERE to END COPY HERE with your mouse
  • Copy the content (e.g. by pressing Ctrl+C)
  • Switch to your own page and place the text cursor where you would like to add the selected element
  • Paste the content (e.g by pressing Ctrl+P)
  • Delete the START COPY HERE and END COPY HERE lines of text
